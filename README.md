# Where Am I?

A small app that shows your location based on your coordinates.

## Live Demo
https://where-am-i-mihailfrunze.netlify.app/

## GitHub
https://github.com/mihail-frunze/where-am-i

## How it works

- The app gets your current position (latitude and longitude)
- It sends the coordinates to a geocoding API
- The API returns your country and city
- The result is displayed on the page
- Country data is also fetched from another API

## What I practised

- Working with APIs (fetch)
- Promises and async/await
- Error handling
- Geolocation API
- Chaining API calls
- Updating the UI with async data
